# PPT模板XML使用指南

## 概述

templates目录包含7个XML模板文件，用于定义不同类型的PPT幻灯片布局。每个模板都遵循Office Open XML (OOXML)格式，定义了幻灯片的结构、占位符位置和样式。

## 模板列表

### 1. title.xml - 标题页模板
**用途**: PPT的首页，用于展示演示文稿的标题和副标题

**布局结构**:
- **主标题区域**: 居中显示，字体较大（54pt）
  - 位置: 居中，Y坐标 1500000 EMU
  - 尺寸: 宽 8000000 EMU，高 2000000 EMU
  - 用途: 显示PPT的主标题
  
- **副标题区域**: 主标题下方
  - 位置: Y坐标 3800000 EMU
  - 尺寸: 宽 8000000 EMU，高 1000000 EMU
  - 用途: 显示副标题、作者、日期等信息

**使用场景**:
- PPT的第一张幻灯片
- 章节分隔页（可选）
- 封面页设计

**示例数据结构**:
```xml
<title>人工智能技术发展</title>
<subtitle>2026年度报告</subtitle>
```

### 2. content.xml - 单列内容模板
**用途**: 标准内容页，包含标题和一列内容区域

**布局结构**:
- **标题区域**: 页面顶部
  - 位置: Y坐标 500000 EMU
  - 尺寸: 宽 8000000 EMU，高 800000 EMU
  - 字体: 32pt
  
- **内容区域**: 标题下方的主要内容区
  - 位置: Y坐标 1400000 EMU
  - 尺寸: 宽 8000000 EMU，高 3800000 EMU
  - 用途: 显示文本、要点列表、段落

**使用场景**:
- 概念说明页
- 要点列表页
- 文字内容为主的幻灯片
- 单列布局的介绍页

**最适合的内容类型**:
- 3-7个要点的列表
- 简短的段落说明
- 单一主题的详细阐述

### 6. section.xml - 章节分隔模板
**用途**: 用于PPT的章节分隔，标记新章节的开始

**布局结构**:
- **章节标题**: 醒目的大标题
  - 位置: 居中偏上
  - 字体: 48pt，粗体
  
- **章节说明**: 标题下方的简短描述
  - 位置: 标题下方
  - 字体: 24pt
  - 用途: 章节概述或引导语

**使用场景**:
- 多章节PPT的分隔页
- 从一个主题过渡到另一个主题
- 提供视觉暂停和结构感
- 帮助观众理解PPT结构

**设计建议**:
- 使用对比色突出章节标题
- 保持简洁，不要过多文字
- 可以加入章节编号

## 坐标系统说明

### EMU单位
所有模板使用EMU (English Metric Units)作为度量单位：
- **1 EMU = 1/914400 英寸**
- **1 英寸 = 914400 EMU**
- **1 厘米 ≈ 360000 EMU**

### 标准PPT尺寸 (16:9)
- **宽度**: 9144000 EMU (10英寸)
- **高度**: 5143500 EMU (5.625英寸)

### 常用尺寸参考
```
标题高度:     800000 EMU  (约0.87英寸)
内容区高度:   3800000 EMU (约4.15英寸)
边距:         1000000 EMU (约1.09英寸)
内容宽度:     8000000 EMU (约8.75英寸)
半宽内容:     3600000 EMU (约3.94英寸)
```

## 使用示例

### Python代码示例

```python
from pptx import Presentation
from pathlib import Path

# 读取模板
template_path = Path('templates/content.xml')

# 创建PPT
prs = Presentation()

# 添加使用content模板的幻灯片
slide = prs.slides.add_slide(prs.slide_layouts[1])  # content布局

# 设置标题和内容
title = slide.shapes.title
content = slide.placeholders[1]

title.text = "机器学习基本概念"
content.text = """
• 监督学习与无监督学习
• 特征与标签的定义
• 数据集的构建与划分
"""

# 保存
prs.save('output.pptx')
```

## 最佳实践

### 1. 模板选择原则
- **内容为王**: 根据内容类型选择模板，不要强行套用
- **保持一致**: 同类内容使用同一模板
- **避免过度**: 不要在一个PPT中使用所有模板

### 2. 布局使用技巧
- **标题简洁**: 控制在1-2行
- **内容适量**: 每页3-7个要点
- **留白充足**: 不要填满所有空间
- **对齐规范**: 遵循模板的对齐方式

### 3. 文字格式建议
- 标题: 32-36pt
- 正文: 18-24pt
- 注释: 14-16pt
- 行距: 1.2-1.5倍

### 4. 颜色方案
- 使用一致的配色方案
- 主色不超过3种
- 保证文字可读性
- 考虑投影效果

## 模板扩展

如需创建新模板，请遵循以下规范：

1. **文件命名**: 使用小写字母和下划线，如 `new_template.xml`
2. **XML结构**: 保持与现有模板一致的XML命名空间
3. **布局设计**: 遵循16:9比例，使用EMU单位
4. **占位符**: 明确定义每个占位符的类型和用途
5. **文档说明**: 在本文档中添加新模板的说明

## 常见问题

### Q1: 如何修改模板的颜色？
A: 在XML中修改 `<a:solidFill>` 标签内的颜色值，使用RGB格式。

### Q2: 可以调整占位符的位置吗？
A: 可以，修改 `<a:off>` 标签中的 x 和 y 值来调整位置。

### Q3: 如何添加新的占位符？
A: 复制现有的 `<p:sp>` 块，修改 id、位置和尺寸参数。

### Q4: 模板支持中文吗？
A: 完全支持，XML文件使用UTF-8编码。

### Q5: 可以在模板中添加动画吗？
A: 建议在生成PPT后使用PowerPoint添加动画效果。
